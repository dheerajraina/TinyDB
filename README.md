# TinyDB

Disclaimer: Code is written by someone who is just starting out with C++, so code may be unsafe.



**Steps to run the project**

1) Generate an executable for main.cpp using "g++ -o tinydb main.cpp"
2) Run the executable using "./tinydb"


**Currently supported commands**

1) CREATE DATABASE database_name
2) USE database_name
3) CREATE TABLE table_name(column_name1 column_type1......column_nameN column_typeN)
